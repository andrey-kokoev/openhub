# @openhub2/provider-azure

OpenHub provider for Azure: Cosmos DB, Redis Cache, Blob Storage.

## Installation

```bash
pnpm add @openhub2/provider-azure
```

## Usage

```typescript
import { azureProvider } from '@openhub2/provider-azure'

// Use with your OpenHub runtime
```

## Supported Bindings

- **database**: Azure Cosmos DB
- **kv**: Azure Redis Cache
- **blob**: Azure Blob Storage
